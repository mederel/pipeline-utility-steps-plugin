Transforms the output into a POJO type (LinkedHashMap or ArrayList) before returning it.

By default deactivated (false), and a JSON object (JSONObject or JSONArray from json-lib) is returned.

Can be activated by default by either:

  • set the following system property in your java start command:
    -Dorg.jenkinsci.plugins.pipeline.utility.steps.json.ReadJSONStep.returnPojo=true
  • or define the environment variable PIPELINE_UTILITY_JSON_RETURN_POJOS with value true
